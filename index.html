Intro

Installable via npm install --save interface-connection, it can also be used directly in the browser.

Download

The source is available for download from GitHub. Alternatively, you can install using npm:

$ npm install --save interface-connection

You can then require() interface-connection as normal:

const interfaceConnection = require('interface-connection')

In the Browser

Interface-connection should work in any ES2015 environment out of the box.

Usage:

<script type="text/javascript" src="index.js"></script>

The portable versions of interface-connection, including index.js and index.min.js, are included in the /dist folder. Interface-connection can also be found on unpkg.com under

  • https://unpkg.com/interface-connection/dist/index.min.js
  • https://unpkg.com/interface-connection/dist/index.js
